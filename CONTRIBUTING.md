# Guia de Contribuição

Obrigado por querer contribuir com o **Tech Cheat Sheets**! Sua ajuda é muito bem-vinda e vai tornar este repositório ainda mais útil para desenvolvedores.

---

## Como Contribuir

### 1. Fork o Repositório
Clique em "Fork" no canto superior direito da página para criar uma cópia do repositório no seu GitHub.

### 2. Clone o Repositório Forkado
Clone o repositório para a sua máquina local:
```bash
git clone https://github.com/seu-usuario/tech-cheatsheets.git
```

### 3. Crie uma Branch para sua Contribuição
Escolha um nome descritivo para a branch:
```bash
git checkout -b nova-contribuicao
```

### 4. Adicione ou Edite Cheatsheets e Recursos
Seja claro e direto no conteúdo. Sugestões de melhorias também são ótimas!
- Para cheatsheets, organize os dados de forma simples e fácil de consultar.
- Para recursos, inclua links relevantes com uma pequena descrição.

### 5. Commit suas Mudanças
Certifique-se de criar mensagens de commit claras e objetivas:
```bash
git add .
git commit -m "Adiciona cheatsheet de CSS"
```

### 6. Envie suas Mudanças
```bash
git push origin nova-contribuicao
```

### 7. Abra um Pull Request
Vá até a página do repositório original e clique em "Compare & pull request". Explique o que foi feito e por quê.

---

## Regras Gerais
- Certifique-se de que seu código está limpo e funciona corretamente.
- Siga o estilo de escrita dos arquivos existentes.
- Evite duplicar conteúdo.

---

## Dicas para Cheatsheets
- Seja direto: Inclua apenas o que é realmente útil.
- Use exemplos sempre que possível.
- Evite detalhes excessivamente técnicos.

---

## Agradecimentos
Sua contribuição faz toda a diferença! Obrigado por tornar o Tech Cheat Sheets ainda mais úteil para a comunidade.
